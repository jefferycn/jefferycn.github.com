ssh rsa key not working

嗯,原因很明确,默认的root目录是在tmp里面,重启以后会被清空

```
/tmp/home/root
```

我的解决办法是: 打开jffs, 然后在里头创建一个目录, 用于存放root用户的各种文件, 当然包括ssh的信息了.

```
cd /jffs/

mkdir -p home/root

mount -o bind /jffs/home/root /root
```
